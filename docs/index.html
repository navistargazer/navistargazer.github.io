Deep RL for
Chrome Dino

구글 크롬의 오프라인 공룡 게임(chrome://dino) 환경을 PyTorch 기반 심층 강화학습(D3QN) 모델로 자동화하는 연구 샌드박스입니다.

시작하기 소스코드 보기
python train_agent.py

비전 모듈에 의해 흑백(Grayscale) 및 외곽선 필터가 적용된 게임 인식 화면

핵심 아키텍처

단순한 스크립트가 아닌, 스스로 관찰하고 학습하는 범용 DRL 파이프라인

👁️

순수 비전 기반 인식

메모리 해킹 없이 mss와 OpenCV만으로 게임 요소 위치, 주야간 테마 변경, 타임라인을 파악하여 Canny Edge로 전처리합니다.

🧠

D3QN 알고리즘

DQN의 과대평가 문제를 해결하는 Double Target 구조와 상황(State)과 선택(Advantage)의 가치를 분리 판단하는 Dueling 아키텍처를 결합했습니다.

📦

Hybrid 메모리 재현

단순 랜덤 경험 재현을 넘어, 가장 예측 오차가 컸던 에피소드를 우선 학습(PER)하여 가중치 수렴 속도를 극대화합니다.

⚡

가속 및 모니터링

NVIDIA CUDA / Apple MPS 자동 식별을 통한 장치 가속을 지원하며, TensorBoard와 내부 통계 산출 로직을 통해 분포 차트를 제공합니다.

사용 방법 가이드

명령행 인터페이스를 통해 직관적으로 하이퍼파라미터를 통제할 수 있습니다.

로컬 환경에 프로젝트를 다운로드하고 의존성을 설치합니다.

git clone https://github.com/navistargazer/dino_rl_agent
cd dino_rein_learning
pip install -r requirements.txt

브라우저에 게임을 띄운 뒤 에이전트 반복 학습을 시작합니다. 파라미터를 통해 아키텍처를 변경할 수 있습니다.

# DQN type 3 (D3QN), Buffer Type 2 (Hybrid) 적용하여 1000판 학습
python train_agent.py --dqn_type 3 --buffer_type 2 --num_episodes 1000

탐험률을 0으로 맞추고, 확보한 pt 파일(가중치)의 진정한 퍼포먼스와 생존 통계(CSV)를 검증합니다.

# 단독 모델 검증 수행
python test_agent.py --dqn_type 3 --buffer_type 2 --num_episodes 50

# CSV 통계 기반 시각화 생성
python utils/analysis.py
